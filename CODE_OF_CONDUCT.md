# Contributor Covenant Code of Conduct

> 这是 Contributor Covenant v2.1 的常用裁剪版。完整版见
> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## 我们的承诺

我们作为成员、贡献者和维护者，承诺让本项目和社区成为一个无骚扰的体验，
不论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、
教育、社会经济地位、国籍、外貌、种族、宗教、性取向。

## 我们的标准

促进社区健康的行为：

- 表达同理心和善意
- 尊重不同的意见、观点和经验
- 给予和优雅地接受建设性反馈
- 承担责任并向受影响者道歉
- 关注的是社区整体的最佳利益

不可接受的行为：

- 使用性化语言、图像或带有性意味的关注 / 推进
- 挑衅、侮辱、贬低、人身攻击或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息
- 在专业场景下不合适的其他行为

## 执行

可疑或违规行为请发到 issue 区或邮件项目维护者。所有投诉会被
及时、公正地审查和调查。维护团队有义务尊重举报人的隐私和安全。
