Lesson 07 · SFT Baseline

目标：用真实 Hugging Face Qwen 0.5B + LoRA，在本地训练一个客服工单到严格 JSON 的 SFT baseline。

数据data/train.jsonl
40 条 instruction/input/output。
评估data/eval_prompts.jsonl
固定 6 条 prompt，对比训练前后。
模型Qwen/Qwen2.5-0.5B-Instruct
按本机 32GB/MPS 选择。
产物outputs/adapter/
adapter、generations、metrics。

执行命令

.venv/bin/python lessons/07-sft-baseline/run.py --trace-delay 0.5

训练流程

01选择 HF 模型
02加载 tokenizer/base
03构造 SFT labels
04训练前生成
05挂 LoRA adapter
06Trainer SFT
07训练后生成
08保存并重新加载 adapter

关键概念

SFT 是用标准答案训练模型。这里标准答案是严格 JSON。

loss mask 是让 prompt 的 labels 等于 -100，只让 answer token 计算 loss。

LoRA 是参数高效训练方式，本课为了在本地跑 Qwen，只训练 adapter，不更新 base model。

adapter 是训练后的增量权重，部署时用同一个 base model 加载 adapter。

看什么结果

  • 打开 report.md 看训练前后 JSON 格式指标。
  • 打开 before.jsonl 和 after.jsonl 看固定 prompt 输出。
  • 打开可视化页看 visualizer/traces/07-sft-baseline.json 的数据流和模型变化。
