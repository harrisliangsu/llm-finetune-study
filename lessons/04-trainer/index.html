Trainer
把数据、模型和 loss 接成闭环

这一课用 tiny causal LM 跑最小训练。它不是可用大模型，目标是看清 Trainer 如何消费 tokenized dataset、计算 loss、更新参数和评估。

3.727训练后 train loss
5.935训练后 eval loss
怎么解释

训练 loss 下降，说明模型确实在训练样本上学习到了 answer token。验证 loss 上升，说明 4 条训练样本太少，模型开始记训练集，泛化变差。

MODEL UPDATE FLOW

tiny causal LM 训练时，真正变化的是模型权重

01 batch
tokenized batch

Dataset 取出一批 input_ids 和 labels，labels 通常是右移后的下一个 token 目标。

[B, T]
02 forward
logits / loss

当前权重做前向计算，得到每个位置对 vocab 的预测分布，再和 labels 算交叉熵。

loss scalar
03 backward
gradients

loss.backward() 把误差沿计算图反传，给每个可训练参数累计梯度。

param.grad
04 optimize
optimizer step

AdamW 读取梯度和学习率，计算每个参数张量应该移动的方向与幅度。

lr=5e-3
05 update
weights updated

参数值被原地更新，下一批数据 forward 时使用的已经是新模型。

all trainable
step 前：旧的 tiny model

embedding.weight / blocks.* / lm_head.weight
这些参数共同决定 logits。

->
step 后：同一模型，权重数值变了

param = param - update
下一轮 loss 会基于更新后的全量参数重新计算。

和 LoRA 不同：Lesson 04 训练的是 tiny causal LM 本体，87,872 个 trainable params 全部参与更新；LoRA 通常冻结 base model，只训练额外插入的低秩 adapter 参数。

GLOSSARY

专有词解释 + 简短例子

Trainer
训练调度器，把 dataset、model、参数配置、评估和保存接成一个训练闭环。
例子：`Trainer.train()` 跑 30 step，最后得到 `train_loss=3.7273`。
forward
模型用当前权重从输入算输出的过程，通常返回 logits，也可能返回 loss。
例子：`input_ids` 进入 tiny causal LM，输出每个位置对词表的预测。
logits
softmax 前的原始预测分数，用来表示模型更偏向哪个候选 token。
例子：token 42 分数 3.1、token 7 分数 0.4，模型更偏向 token 42。
loss
预测和正确答案之间的差距，是训练要不断降低的目标。
例子：训练后 `train loss=3.7273`，说明训练样本上的预测变好了。
cross entropy
常用分类 loss，正确 token 概率越低，惩罚越大。
例子：正确答案是“累”，模型给“累”的概率越高，cross entropy 越低。
backward
从 loss 反向计算梯度，告诉每个参数应该怎样调整才能降低 loss。
例子：`loss.backward()` 后，模型参数的 `.grad` 会被填上数值。
gradient
loss 对参数的变化方向和大小，是 optimizer 更新参数的依据。
例子：某个权重 gradient 很大，说明它对当前 batch 的 loss 很敏感。
optimizer
根据 gradient 修改参数的算法，把反向传播结果变成真正的权重更新。
例子：AdamW 读取 `.grad`，结合学习率更新 `lm_head.weight`。
learning rate
每次参数更新的步子大小，决定学习速度和稳定性。
例子：本课 `lr=5e-3`；loss 乱跳时通常先尝试调小。
epoch
完整看完一遍训练集的次数，用来描述数据被重复学习了多少轮。
例子：4 条训练样本、batch size 2，跑 2 个 batch 就是 1 个 epoch。
overfitting
模型更会记训练集，但对没见过的数据变差，需要看验证集发现。
例子：本课 train loss 下降，但 eval loss 升到 `5.9352`，是过拟合信号。
checkpoint
训练中保存的模型状态，用于恢复训练、回滚版本或做中间评估。
例子：真实训练常保存 `checkpoint-100`；本课输出目录是 `lessons/04-trainer/outputs/trainer`。
